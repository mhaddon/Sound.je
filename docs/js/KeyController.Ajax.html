Class: Ajax

Core/KeyController. Ajax

new Ajax(url, method, data)

A simple AJAX class that creates an AJAX request and has jquery-like methods for handling the response. This class is wrapped in Util.ajax For example: const ajaxRequest = new Ajax(url); ajaxRequest.onSuccess(callbackFunction).onFailure(callbackFunction).onComplete(callbackFunction);
Parameters:
Name Type Argument Default Description
url string <optional>
 '/' ajax endpoint
method string <optional>
 'get' http request method to use, get, post, put, delete....
data Object <optional>
 {} if we need to send additional data, like a post, then the data goes in here
