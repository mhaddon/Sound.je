Class: Listener

Listener

new Listener(active, el, binding)

This class is responsible for controlling an infinite scroller listener The listener waits until it hears the signal that its parent element has scrolled to the bottom Then it loads any extra data it wants
Parameters:
Name Type Description
active boolean Whether or not the listener is active
el Element dom element the listener refers to
binding ListenerBinding v-infinite binding
