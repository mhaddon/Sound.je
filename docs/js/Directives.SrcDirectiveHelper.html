setBackground(el, background, id)

Higher order function that sets the background of an image. If no background is detected then it defaults to the HOF background Sets the src tag to the URL of the downloaded image and resets the background to the state it was in prior to this method running
Parameters:
Name Type Description
el Element element we are changing
background string the original background
id string the id of this request
