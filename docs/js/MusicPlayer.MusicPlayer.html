Class: MusicPlayer

MusicPlayer

new MusicPlayer()

Singleton class that controls the music player on the page. The majority of the state for the music player is stored in the vuex store so vue can make use of the reactive variables. The main purpose of this class is to create a common interface for controlling the music players of many different APIs.

media :Object

Currently loaded media element This could be a Facebook video element, Soundcloud song element, or a Youtube video element. Meaning the contents of this element can change drastically. The only constant behaviour is that all the media elements have a targetElement field which links the object to the HTML block that is responsible for it existing. This targetElement field is added by us, so it may not be in the documentation.
